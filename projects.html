To-do List

To-do List is a project that I developed to practice some React concepts, such as 'useState' and 'useEffect'.

This project allows users to add, edit, remove, and mark tasks as completed. The user can also delete all items, to avoid unnecessary repetitive work.

The board is designed to be expandable, so it has a default size, but if the user adds more items than the default size, the board expands accordingly while the background image remains fixed, only task board scrolls.

The latest funcionality I added to this project is local storage. This ensures that if the user reloads the page, the tasks will still be saved.

DescarTech

DescarTech is an app developed using React Native with the objective of reducing e-wast by reusing old devices or showing the places to discard them.

There are some important screens:

1. Welcome page: users can log in or create a new account.

2. Feed page: users can see the devices and their descriptions.

3. Product details page: users can see more information and pictures about the device.

4. User profile page: users can see all the devices posted by a particular user.

5. Map page: users can see the appropriate places to discard these kinds of devices. This page uses the Google Maps API and while the response is loading, a loader is displayed on the screen

At this point, mocked data is being used.
